Vroom Vroom — Architecture Defense

AI harness for ICE vehicle DIY diagnosis · $0 stack · Vercel Hobby

Problem

Raw LLM chat hallucinates TSB numbers, recommends dangerous procedures, and cites unverified forums. Vroom Vroom grounds answers in verifiable data and fails safely when confidence is low.

Four Pillars

PillarRole
LoopConstrained agent: max 5 tool rounds, 60s, 2 web searches
ToolsKB → NHTSA VIN → Tavily (domain allowlist)
GuardrailsPrompt rules, Zod schema, blocklist, safe rewrite
ObservabilityJSON logs + UI trace panel

Key Decisions

  • Next.js + TypeScript — typed tools, streaming API, Vercel deploy
  • Gemini Flash — free tier; tools do heavy lifting
  • Tavily allowlist — server-side domain lock
  • Markdown export — no DB, client-side download

Deliberately Not Built

  • Unrestricted web search
  • Paid APIs
  • EV/hybrid HV diagnostics
  • Multi-agent swarms

Tradeoffs

Tavily quota (~500 diagnoses/mo) is the capacity ceiling. Diagnoses degrade to offline KB when exhausted — correct behavior. Flash is less capable than Pro, but tool-grounded reasoning compensates.

Conclusion

Vroom Vroom is an AI harness, not a chatbot. Safety, verifiability, and zero cost beat maximal intelligence for untrained users working on their own cars.
